# oshaberu_clova

## env

- DOMAIN : 音声などが置かれているドメイン

- EXTENSION_ID : ExtensionのID

## responses.json

```json
{
    // 単純なランダム
    "Intent名1" : [
        "セリフ1", "セリフ2", "セリフ3",...
    ],

    "Intent名2" : {
        // Slot値に応じる
        "Slot名1" : {
            "Slot値1" : [
                "セリフ1", "セリフ2", "セリフ3",...
            ],
            "Slot値2" : [
                "セリフ1", "セリフ2", "セリフ3",...
            ]
        },
        "Slot名2" : {
            ...
        },

        // その他
        "" : [
            "セリフ1", "セリフ2", "セリフ3",...
        ]
    },

    // その他
    "" : [
        "セリフ1", "セリフ2", "セリフ3",...
    ]
}
```
